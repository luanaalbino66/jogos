# Jogos Eletrônicos - Busca Rápida

## Sobre
Este projeto web permite aos usuários pesquisarem por jogos eletrônicos em uma lista pré-definida. Ao digitar o nome de um jogo no campo de pesquisa, a aplicação retorna uma lista de jogos relevantes, exibindo título, descrição e link para mais informações.

## Tecnologias Utilizadas
* **HTML:** Estruturação da página.
* **CSS:** Estilização da página.
* **JavaScript:** Lógica da aplicação, incluindo a função de pesquisa.

## Como Funciona
1. **Pesquisa:** O usuário digita o nome de um jogo no campo de pesquisa.
2. **Processamento:** O JavaScript compara a entrada do usuário com os títulos, descrições e tags de cada jogo na lista de dados.
3. **Exibição:** Os resultados da pesquisa são exibidos na seção de resultados, mostrando o título, uma breve descrição e um link para mais informações.

## Estrutura dos Dados
Os dados dos jogos são armazenados em um arquivo JavaScript separado (dados.js). Cada jogo é representado por um objeto com as seguintes propriedades:
* `titulo`: Título do jogo.
* `descricao`: Descrição curta do jogo.
* `link`: Link para mais informações sobre o jogo.
* `tags`: Palavras-chave relacionadas ao jogo.
